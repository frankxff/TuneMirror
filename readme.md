# 音准照妖镜 - TuneMirror

音准照妖镜是一款本地离线音准分析工具，用于对纯人声音频进行音高检测、音准偏差统计和可视化对比。项目包含通用分析页和“歌手2026专题”展示页。

## 功能特性

- 支持上传一个或多个音频文件进行本地分析
- 展示音准偏离度分布、综合得分、完美率、中位偏差、P80/P95 偏差等指标
- 支持历史记录与多音频对比
- 提供修音概率与假唱概率的统计参考
- 包含“歌手2026专题”页面，按期展示分析结果

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。

如需查看专题页面，可打开 `singer2026.html`，或从首页点击“歌手2026专题”入口。

## 文件说明

- `index.html`：音频上传、分析、历史记录与对比主页面
- `singer2026.html`：歌手2026专题分析展示页面
- `echarts.min.js`：图表渲染依赖文件，需要与 HTML 文件放在同一目录

## 注意事项

本项目的音准得分、修音概率、假唱概率会受音源质量、剪辑片段、伴奏、混响、和声、录制设备、现场收音和算法误差影响，仅供娱乐和技术参考，不代表任何官方结论，也不可作为评价歌手真实演唱能力、修音或假唱的最终证据。
